## 주요 진행 사항

### ✅ 완료된 작업들
1. **포트폴리오 롤백** - emoji 디자인으로 변경
2. **API 수정** - 업비트 직접 호출로 프록시 제거
3. **반응형 디자인** - 모든 CSS 파일에 모바일 최적화 추가
4. **차트 통일** - CandleChartLW(ApexCharts) 사용
5. **볼륨 차트** - Bar → Line 변경 (자주색)
6. **BTC 도미넌스 제거** - 헤더에서 비트코인 도미넌스 바 제거
7. **찐콘 교체** - favicon을 imdol-logo.png로 변경
8. **모바일 최적화** - 최근 검색어, 코인 심볼 폰트 크기 조정
9. **대시보드 점수 통일** - crypto-dashboard 버전에 점수 표시 기능 추가

## 기술 결정사항
- **추천 시스템**: `getTodayRecommendations()` 유틸로 일일 추천 관리
- **점수 표시**: 80점 이상 = 금색 하이라이트, 그 외 = 기본색
- **스타일링**: CSS 모듈 + 인라인 스타일 (조건부)
- **갱신 주기**: 매 5초마다 추천 데이터 리로드
