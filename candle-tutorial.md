# 🕯️ 캔들 차트 완전 초보 가이드

## 1. 캔들 하나 = 하루 (또는 한 시간, 한 주)

```
         ↑ 고가 (High) - 그날 가장 높았던 가격
         │
     ┌───┴───┐
     │   ↑   │  ← 몸통 (Body)
     │       │
     └───────┘
         │
         ↓ 저가 (Low) - 그날 가장 낮았던 가격
```

---

## 2. 빨간 캔들 = 상승한 날 📈

```
        고가 35,000원
          │
          │ ← 위 꼬리 (상승 시도했다가 내려옴)
      ┌───┴───┐
      │  빨강  │ ← 몸통 (시가 → 종가 상승)
      │       │
      │ 시가  │    시가 28,000원
      └───────┘    종가 32,000원
          │
          │ ← 아래 꼬리 (하락했다가 다시 올라옴)
        저가 27,000원
```

**읽는 법:**
- 시작: 28,000원
- 최저: 27,000원 (떨어졌다가)
- 최고: 35,000원 (올라갔다가)
- 끝: 32,000원 마감
- **결과: +4,000원 상승!** 🔥

---

## 3. 파란 캔들 = 하락한 날 📉

```
        고가 33,000원
          │
          │ ← 위 꼬리 (올라갔다가 떨어짐)
      ┌───┴───┐
      │ 시가  │    시가 32,000원
      │       │    종가 28,000원
      │  파랑  │ ← 몸통 (시가 → 종가 하락)
      └───────┘
          │
          │ ← 아래 꼬리 (더 떨어졌다가 조금 회복)
        저가 26,000원
```

**읽는 법:**
- 시작: 32,000원
- 최고: 33,000원 (올라갔다가)
- 최저: 26,000원 (떨어졌다가)
- 끝: 28,000원 마감
- **결과: -4,000원 하락!** 📉

---

## 4. 꼬리(그림자)의 의미

### **긴 아래 꼬리 = 강세 신호!** 🟢

```
      ┌───┐
      │   │
      └───┘
         │
         │ ← 긴 아래 꼬리!
         │
         │
        ╰──
```

**의미:**
- 장중에 **많이 떨어졌다가**
- **다시 올라와서** 마감
- = **매수 세력이 강하다!** 💪
- 바닥에서 반등할 가능성 높음

---

### **긴 위 꼬리 = 약세 신호!** 🔴

```
        ╭──
         │
         │ ← 긴 위 꼬리!
         │
      ┌──┴──┐
      │     │
      └─────┘
```

**의미:**
- 장중에 **많이 올랐다가**
- **다시 떨어져서** 마감
- = **매도 압력이 강하다!** 📉
- 고점에서 저항 받는 중

---

## 5. 실전 예시: 삼성중공업 우측 끝

```
오늘 캔들 (2/20):
        
        30,000원 ← 고가
          │
      ┌───┴───┐
      │ 빨강  │  시가 29,000원
      │       │  종가 29,450원
      └───────┘
          │
        28,800원 ← 저가
```

**해석:**
- 시가 29,000 → 저가 28,800 (아침에 떨어짐)
- → 다시 올라서 고가 30,000 터치
- → 29,450으로 마감
- **결과: +1.90% 상승** ✅
- 아래 꼬리 짧음 = 매수 세력 보통

## 6. 패턴 읽기

### **좋은 신호들:**

```
연속 양봉 (3일 연속):
┌──┐ ┌──┐ ┌──┐
│빨│ │빨│ │빨│  ← 계속 올라가는 중!
└──┘ └──┘ └──┘
```

```
긴 아래 꼬리 (바닥 다짐):
     ┌─┐
     └─┘
       │
       │  ← "여기가 바닥이다!"
       │
      ╰─
```

### **나쁜 신호들:**

```

┌──┐ ┌──┐ ┌──┐
│파│ │파│ │파│  ← 계속 떨어지는 중...
└──┘ └──┘ └──┘
```

```
긴 위 꼬리 (고점 거부):
      ╭─
       │
       │  ← "더 못 올라가!"
     ┌─┴─┐
     │   │
     └───┘
```

## 7. 삼성중공업 현재 상황 요약

```
차트 전체:

33,000 ─────────────────── ← 저항선 (못 뚫음)
                          
30,000 ═══════════════════ ← 현재 도전 중!

29,450 ★ ← 지금 여기 (오늘)

27,000 ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ ← 강한 지지선 (3번 막음)

24,500 ─────────────────── ← 바닥
```

**결론:**
- 27,000에서 **3번 튕김** = 바닥 확인 ✅
- 이제 30,000-33,000 돌파 시도 중
- 성공하면 → 35,000+ 가능
- 실패하면 → 27,000 재테스트

## 🎯 주린이를 위한 실전 전략

### **지금 삼성중공업 (29,450원):**

**시나리오 1: 물타기 (공격적)**
```
지금 100주 추가 @ 29,000원
→ 평단 30,000 → 29,500원
→ 30,000원만 와도 본전!
→ 33,000원 가면 +11% 수익
```

**시나리오 2: 지지 확인 매수 (안전)**
```
27,000원 터치 시 200주 추가
→ 평단 28,000원대
→ 30,000원 오면 +7% 수익
→ 리스크: 안 내려올 수도
```

**시나리오 3: 손절 (정리)**
```
지금 전량 매도 @ 29,450원
→ -5.5만원 손실로 마무리
→ 자금 295만원 회수
→ 더 확실한 곳(SKT, 한화오션)에 재투자
```

## 💡 돌돌이 최종 추천

**Option C: 손절 후 한화오션 집중**

**이유:**
1. 손실 미미 (-5.5만 = 커피 10잔 값)
2. 한화오션이 삼성중공업보다 **모든 면에서 우위**
   - 방산 + 조선 (vs 조선만)
   - 수주액 1위
   - 차트도 더 강함
3. 시드 있으니까 분산보단 **집중이 유리**

**액션:**
```
□ 삼성중공업 100주 매도 → 295만 회수
□ 한화오션 2주 추가 (28만)
□ SK텔레콤 3주 추가 (250만)
□ 현금 17만원
```

## 📚 캔들 공부 요약

**기억할 것 3가지:**
1. **빨강 = 올랐다 / 파랑 = 떨어졌다**
2. **긴 아래 꼬리 = 좋은 신호** (매수 세력 강함)
3. **긴 위 꼬리 = 나쁜 신호** (매도 압력 강함)
